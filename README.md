# 🚀 Rizz Master

AI-powered conversation coach designed to help users create better conversations, build confidence, and improve their chances of turning matches into real-life dates.

---

## 📱 About

Rizz Master is a Flutter application powered by Artificial Intelligence that analyzes conversations and profiles to generate personalized response suggestions.

The goal is simple:

* Start better conversations
* Keep conversations engaging
* Build attraction naturally
* Help users move from chat to real-world meetings

## 🔒 Privacy

User screenshots are never stored permanently.

Workflow:

1. User uploads a screenshot.
2. AI analyzes the content.
3. Analysis result is saved.
4. Screenshot is automatically deleted.

Only conversation insights and generated suggestions are stored.
